Research

Nov. 11, 2024

QT-WEAVER

Overview

Summarizing individual gene trees into species phylogenies using coalescent-based methods has become a standard approach in phylogenomics. However, gene tree estimation error (GTEE)—arising from both analytical factors and biological causes (e.g., short gene sequences)—can adversely impact the accuracy of phylogenomic inference.QT-WEAVER introduces the novel problem of correcting the quartet distribution induced by a set of estimated gene trees.

Jul. 31, 2024

WQFM-tree

Overview

wQFM-TREE is a summary method developed for species tree estimation from multi-locus data, addressing gene tree discordance. It builds on the wQFM method and improves its scalability, making it applicable to large datasets.

CHIP-8 Emulator

This project is a CHIP-8 emulator written in Golang. The Ebiten game engine was used to create the graphical interface. Developing this project helped me understand how emulators work by simulating the behavior of a physical machine in software.

Yet Another C Compiler

This repository contains my implementation of a C Compiler, developed as part of a lab project during my undergraduate studies at BUET. The project encompasses the complete pipeline of compiler construction, progressing through various stages to produce a functional compiler.

xv6 Operating System Enhancements

This project extends the xv6 Operating System, a simple Unix-like OS developed for educational purposes. The enhancements include the implementation of advanced scheduling algorithms, paging, and file system features. The project was initially developed as part of an operating systems lab, showcasing practical skills in OS development.
