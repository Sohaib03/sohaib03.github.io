CHIP-8 Emulator

This project is a CHIP-8 emulator written in Golang. The Ebiten game engine was used to create the graphical interface. Developing this project helped me understand how emulators work by simulating the behavior of a physical machine in software.

About CHIP-8

  • CHIP-8 is an old, simple programming language designed for running games in the 1970s.
  • It supports 35 instructions (opcodes), which control tasks like drawing, memory operations, and branching.
  • The system has 16 registers, a 4K memory space, and a stack for handling subroutines.
  • The graphics are black-and-white with a resolution of 64x32 pixels, suitable for basic games.

Project Highlights

  • Implemented Opcodes: Executed all CHIP-8 instructions to run programs correctly.
  • Simulated Hardware: Emulated memory, registers, and the stack for CHIP-8 programs.
  • Graphical Output: Used Ebiten to display the 64x32 monochrome grid.
  • Input Handling: Added support for the CHIP-8 16-key hexadecimal keypad.

Summary

This project introduced me to emulator development by working on a small and simple system. It gave me a clear idea of how instructions, memory, and graphics interact in a machine.
